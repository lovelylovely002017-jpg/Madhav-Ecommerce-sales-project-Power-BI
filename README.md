# Madhav Ecommerce Sales Dashboard

## Project Overview

This Power BI dashboard provides a comprehensive analysis of Madhav Ecommerce sales data. The dashboard helps track sales performance, profit trends, customer behavior, and product category performance through interactive visualizations.

## Key Insights

* Total Sales and Profit Analysis
* State-wise Sales Performance
* Category and Sub-Category Analysis
* Customer-wise Revenue Contribution
* Monthly Profit Trends
* Payment Mode Distribution
* Interactive Filters and Slicers

## Tools Used

* Power BI
* DAX
* Data Modeling
* Data Visualization

## Business Value

The dashboard enables stakeholders to monitor key business metrics, identify top-performing products and regions, and make data-driven decisions to improve sales performance.
